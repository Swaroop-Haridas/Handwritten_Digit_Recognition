# Handwritten Digit Recognition

A deep learning-based handwritten digit recognition system built using **Python**, **TensorFlow/Keras**, and the **MNIST dataset**. This project demonstrates the complete workflow of image classification, including data preprocessing, neural network training, model evaluation, and digit prediction.

---

## 📌 Project Overview

Handwritten digit recognition is a fundamental computer vision task with applications in postal mail sorting, bank cheque processing, document digitization, and optical character recognition (OCR). This project implements a neural network that learns to classify grayscale images of handwritten digits into one of ten classes (0–9) using the widely used MNIST dataset.

The notebook walks through the complete machine learning pipeline, making it suitable for beginners learning deep learning concepts as well as those exploring image classification using TensorFlow.

## 📂 Dataset

This project uses the **MNIST Handwritten Digit Dataset**, one of the most widely used benchmark datasets for image classification.

Dataset characteristics:

- 60,000 training images
- 10,000 testing images
- Grayscale images
- Image size: **28 × 28 pixels**
- 10 output classes (Digits **0–9**)
